# ADR-010 — Combobox для выбора цели KPI (@headlessui/react)

- **Статус:** принято
- **Дата:** 2026-05-27
- **Контекст:** fix-prompt FINAL §9.2 требует заменить `<select>` GOAL_ID на combobox с поиском и
  группировкой (Активные/Архивные). У счётчика десятки целей — нативный select неудобен.
  По `CLAUDE.md` новая зависимость — через ADR.

## Решение

Добавить **`@headlessui/react`** и использовать его `Combobox` для выбора цели KPI в Настройках.

## Обоснование

- Доступный по умолчанию (ARIA, клавиатура), unstyled — стилизуется Tailwind в едином стиле.
- Поиск + произвольный рендер опций (группы Активные/Архивные) из коробки.
- От тех же авторов, что Tailwind; зрелый, MIT. Альтернатива (свой combobox) — много a11y-кода.

## Последствия

- Combobox реализован инлайн в `routes/settings.tsx` (страница вне coverage — как и весь
  Settings c таймерами/мутациями; покрывается e2e smoke). Текстовый поиск + группы.
- Дальнейшие комбобоксы/диалоги — из этой же библиотеки.
