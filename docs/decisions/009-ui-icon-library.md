# ADR-009 — Библиотека иконок интерфейса (lucide-react)

- **Статус:** принято
- **Дата:** 2026-05-27
- **Контекст:** fix-prompt FINAL §7.5/§10.4 требует иконки в навигации (и в перспективе по UI).
  По `CLAUDE.md` зависимость вне зафиксированного стека добавляется только через ADR.

## Решение

Добавить **`lucide-react`** как зависимость фронтенда для иконок навигации и UI.

## Обоснование

- Tree-shakeable (импортируются только используемые иконки) — минимальный вклад в бандл.
- React-компоненты, типобезопасны, `aria-hidden` по умолчанию декоративны.
- Зрелая, активная, MIT-лицензия; де-факто стандарт в React-экосистеме (форк Feather).
- Альтернатива (инлайн-SVG вручную) — больше шаблонного кода и риск рассинхрона стилей.

## Последствия

- Иконки в `Layout.tsx` (9 пунктов навигации), декоративные (`aria-hidden`), текст пунктов остаётся
  основным лейблом (доступность не зависит от иконки).
- Дальнейшее использование иконок — из этой же библиотеки (единый визуальный язык).
