## 🔴 Открытые дефекты v2.8.3

### M-004 [Major] — UTM-coverage 0.0% подозрительно

- **Где:** /overview бейдж «Низкое покрытие UTM: 0% (порог 70%)», /traffic тот же бейдж, превью отчёта тоже.
- **Контекст:** UTM-таблица заполнена (tg/infopartner/codedaria/zamesin_flow), значит utm_stats содержит строки. Формула `utmCoverage` считает не то.
- **Фикс:** см. fix-prompt-FINAL.md §5.

### M-DET [Major] — DOCX/PDF SHA-256 не детерминистичны между генерациями

- **Симптом:** одна и та же snapshot dd034e85 → 2 генерации → разные SHA-256.
- **Причина:** zip/pdf metadata содержат timestamps (исключения «known limit» в CLAUDE.md).
- **Контент детерминистичен:** PNG-имена-хэши одинаковые.
- **Решение:** либо явно зафиксировать в спеке («content SHA-256, не файловый»), либо запекать UTC=0 в zip/pdf creation timestamp.

### C-005 [Minor] — Дубли URL в «Топ страниц входа»

- **Где:** /overview блок «Топ страниц входа»: 4 строки `https://productcamp.ru/` с разными визитами/bounce/заявками.
- **Старый дефект из v2.3.0** (фиксировался, но всё ещё виден).
- **Причина:** URL не нормализованы (фрагменты/query треkenы как разные страницы).

### m-007 [Minor] — GOAL_ID — обычный input, не combobox с поиском

- **Где:** /settings «Цель KPI (GOAL_ID)».
- **Контекст:** в release notes v2.8.x заявлен «GOAL_ID combobox с поиском и группами Активные/Архивные (@headlessui)», но видно обычный input number=2.
- **Возможно** combobox появляется иначе (другой view), либо ещё не реализовано полностью.

### m-snapshot-label [Minor] — В KPI снапшота лейбл «Заявки B2C» при e_purchase

- **Где:** /report превью KPI-блок «ЗАЯВКИ B2C 50 · заявка ≠ оплата».
- **Контекст:** на /overview и /goals лейбл «Оплат» (formatGoalLabel применён), в снапшоте — «Заявки B2C». Несогласованность.

### m-api-key [Minor] — Anthropic API Key показан пустым placeholder в /settings

- **Где:** /settings «Anthropic API Key» — placeholder sk-ant-..., пусто.
- **Контекст:** `/api/settings` отдаёт маску `"ANTHROPIC_API_KEY":"sk-a****AA"`, значит ключ в .env есть. UI должен показывать маску в input, не пустое поле.
